## Uso con Claude Code

Copiar el MASTER_PROMPT como memoria global:

```bash
cp NEWCOOL_MASTER_PROMPT.md ~/.claude/CLAUDE.md
```

O usar como referencia en proyecto:

```bash
cp NEWCOOL_MASTER_PROMPT.md .claude/CLAUDE.md
```
